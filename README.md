# Linux Automation Suite

A collection of automated Bash scripts to keep a Linux server healthy.

## Scripts
- disk_check.sh — alerts when disk usage exceeds 80%
- log_cleanup.sh — rotates and compresses logs older than 7 days
- user_audit.sh — weekly report of sudo users and recent logins
- backup.sh — daily backup of critical directories
